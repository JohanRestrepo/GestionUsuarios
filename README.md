# proyecto de prueba tecnica (falle pero lo construyo igual)

# Para hacer funcionar este proyecto debes instalar

npm install react-hook-form -> con esto nos facilitamos el trabajo de los formularios
npm install react-router-dom -> con esto nos aseguramos que react cambie de rutas
npm install react-icons --save -> con esto instalamos los iconos que uso en la app -> https://react-icons.github.io/react-icons

# Para hacerlo correr debes usar

`npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000)
